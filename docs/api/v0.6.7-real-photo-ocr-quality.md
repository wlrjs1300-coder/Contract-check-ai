# v0.6.7 실제 촬영 계약서 사진 OCR 품질 강화 계약

## 1. 적용 범위

- 브랜치: `feat/v0.6.7-real-photo-ocr-quality`
- 대상 엔드포인트: `POST /extractions/images`, `GET /extractions/{id}`, `GET /extractions/{id}/review`, `PATCH /extractions/{id}/pages/{page_id}/review`, `POST /extractions/{id}/confirmation`
- 대상 파일 형식: PNG/JPEG/JPG (확장자, MIME, signature, decoder 동시 검증)
- 목표: 실제 촬영 계약서 사진(기울기·그림자·노출·흐림·저해상도·잘림 등)에서 품질 경고를 안정적으로 노출해 사용자 결정 지원
- 금지: 실제 OCR 결과를 자동 신뢰로 처리하거나 성공 판정으로 위장하지 않음

## 2. 품질 경고 코드

- `low_resolution` / `image_resolution_low`
- `possible_blur_or_blank`
- `image_blurry`
- `image_quality_too_low`
- `image_unreadable`
- `image_underexposed`
- `image_overexposed`
- `image_low_contrast`
- `image_shadow_suspected`
- `image_glare_suspected`
- `image_low_contrast`
- `deskew_uncertain`
- `perspective_distortion_suspected`
- `document_boundary_uncertain`
- `background_included`
- `document_crop_suspected`
- `text_too_small`
- `ocr_amount_format_suspected` / `ocr_date_format_suspected`
- `ocr_confidence_low`, `ocr_low_confidence`

표기 중복은 중복 제거되어 응답/저장 시 노출되지 않는다.

## 3. 자동 보정 정책

- `deskew_not_applied`: 현재 기본 경로에서 무조건 90도 EXIF 정규화만 수행하고 임의 각도 deskew를 미적용
- 보존 정책: 자동 보정이 가능한 경우만 제한적으로 적용(현재 단계는 핵심 품질 경고 산출 중심)
- 미지원 범위: 강한 perspective correction, aggressive sharpen, 자동 그림자 제거, glare 복원은 구현하지 않음
- 미보장: 경고 없음이 정확성 보장으로 해석되지 않음

## 4. 품질 기반 사용자 안내

- 보정·재촬영 시점은 `review_required` 상태 유지
- 심각한 품질 경고(`image_quality_too_low`, `image_unreadable`)는 기존 파이프라인에서 확인 단계로 유지
- 사용자에게 재촬영 권고 문구는 프론트에서 경고군을 기반으로 표시하도록 설계(본 단계 API 계약에서는 코드만 노출)

## 5. 전처리 단계

1) decode/서명/decoder 검증
2) EXIF 정규화
3) alpha 합성
4) 메타데이터 제거
5) 품질 메트릭 산정(내부)
6) 경고 반영
7) OCR 실행
8) warning/리뷰 플래그 조합

raw 메트릭(히스토그램, 모멘트, 스코어)은 API 응답/스키마에 과다 노출하지 않는다.

## 6. 실패/차단 경계

- 현재 단계는 fail-closed: 품질 점수로 `analysis_blocked`를 완화하지 않음
- `image_unreadable` 또는 OCR 예외 발생은 기존 오류 코드 또는 분석 경고로 실패가 추적됨
- 원본/파생 파일 cleanup 실패는 기존 `original_cleanup_failed` 경로를 유지

## 7. 테스트 범위

- 질의 집약적 품질 경고 단위 테스트
  - clean, blur mild/severe, 저해상도, 기울기
  - 경계/배경/잘림 후보
- 기존 이미지 OCR 회귀 + 실제 로컬 OCR 통합 연동 흐름 유지
- OpenAPI 계약 유지
