## 결과

### 변경 파일

### 확정 내용

- MVP는 계약서 업로드, 텍스트 추출, 조항 분리, 위험 가능성 분석, 결과 저장 및 조회 흐름을 중심으로 한다.
- AI 분석 전 개인정보 탐지와 마스킹을 필수 처리 단계로 둔다.
- 분석 진행 중, 완료, 실패 상태 구분을 필수 기능으로 둔다.
- 업로드 파일 지원 여부 확인과 미지원 파일 안내를 MVP 범위에 포함한다.
- 인증은 이메일/비밀번호 기반 회원가입, 로그인, 로그아웃, 본인 데이터 접근으로 제한한다.
- 분석 이력은 본인 분석 목록, 문서 표시명, 생성일, 상태, 완료 결과 재조회로 제한한다.
- 공식 사용자 업로드 지원 파일은 텍스트 레이어가 있는 PDF로 한정한다.
- AI 분석 결과는 관련 문구, 위험 가능성, 확인 이유, 추가 질문, 권장 행동, 전문가 검토 권고 여부를 포함한다.
- 원본 계약서 파일은 영구 저장하지 않고 처리 완료 후 삭제하는 방향으로 둔다.
- 법률 자문, OCR, 관리자 페이지, 전자서명, 계약서 자동 작성, ERP 수준 기능은 MVP에서 제외한다.

### 검증 결과

- 기준 문서와 충돌하지 않는 범위에서 MVP 범위를 작성했다.
- 공개 저장소에 남겨도 되는 정제된 문장으로 작성했다.
- 개인정보, Secret, 원본 계약서, 검토 전 입력/출력 원문은 포함하지 않았다.
- API, DB, UI, AI 프롬프트, 구체적인 보안 수치를 이번 문서에서 임의로 확정하지 않았다.
- 독립 검토 결과의 필수 수정 2건과 권장 수정 4건을 반영했다.
- 최종 경미한 권장 수정 1건을 반영했다.
- 필수 수정 잔여 항목이 없음을 확인했다.
- 최종 Git 검증을 완료했으며 변경 범위가 예정된 문서 3개로 제한되어 있음을 확인했다.
