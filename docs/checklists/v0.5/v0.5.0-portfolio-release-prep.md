## 알려진 한계

- 루트 README는 이번 버전에서 처음 추가되며 기존 공개 첫 화면과의 이전 비교 대상이 없다.
- 지정된 `backend/app/core/config.py`와 `backend/app/db/session.py`는 현재 저장소에 없고 실제 설정은 `backend/app/main.py`와 `backend/app/db/database.py`에 있다.
- 장기 기술 문서는 MySQL, JWT와 실제 Provider 등을 계획으로 포함하지만 현재 구현은 SQLite·무인증·합성 Provider다. 공개 문서는 실제 코드를 우선했다.
- 기존 버전 정책의 초기 계획과 실제 운영 버전 번호가 다르다. 이미 병합된 번호를 유지하고 변경 이력 원칙에 따라 현재 작업을 기록했다.
- 실제 screen reader, 전문 색상 대비와 실제 배포 환경은 검증하지 않았다.
