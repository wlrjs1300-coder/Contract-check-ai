- 외부 AI API 호출
- API Key 준비
- provider adapter 구현
- 공급자 최종 선정
- endpoint 또는 model 최종 선정
- 실제 계약서 테스트
- 실제 개인정보 테스트
- 실제 파일 업로드 구현
- JWT 구현
- 인증 API 구현
- DB schema 구현
- 백그라운드 작업 복구 구현
- retry/backoff 구현
- 배포 Secret 설정
- CI/CD 구현
- 제품 운영 승인
- 프로덕션 출시
- 법률 자문
