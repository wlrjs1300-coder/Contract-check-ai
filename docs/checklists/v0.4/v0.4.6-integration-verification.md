## 발견 결함

### MINOR — 320px 가로 스크롤

- 재현: Chrome viewport 320px에서 문서 B 결과 화면 확인
- 실제: `documentElement.scrollWidth`가 client width보다 큼
- 기대: 320px에서 가로 스크롤 없음
- 원인: `body { min-width: 320px; }`와 세로 스크롤바 폭의 합으로 viewport 초과
- 수정: body 고정 최소 너비 한 줄 제거
- 회귀: 실제 Chrome 320px 재검증과 CSS 회귀 테스트 통과

### OBSERVATION — host와 CORS origin

- `127.0.0.1:5173`은 기본 allowlist에 없으므로 브라우저가 응답을 차단한다.
- 기본 계약은 `http://localhost:5173`이며 해당 origin에서는 완료했다.
- 코드 수정 없음. 다른 host를 사용할 때는 세션 CORS·API URL을 함께 맞춰야 한다.

BLOCKER와 MAJOR 결함은 발견하지 않았다.
