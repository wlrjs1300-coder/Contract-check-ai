## 프론트 scaffold 계약

- React, Vite, TypeScript strict 설정을 사용한다.
- Bootstrap 5와 프로젝트 CSS만 사용한다.
- 초기 화면은 로컬 통합 준비 상태만 표시한다.
- API 전체 client나 실제 분석 기능은 만들지 않는다.
- API 기본 URL은 `VITE_API_BASE_URL`에서 읽고 기본값은 `http://localhost:8000`으로 둔다.
- API URL의 끝 slash를 제거해 경로 중복을 방지한다.
- 브라우저 환경변수에 비밀값을 저장하지 않는다.

---

## CORS 계약

- 기본 허용 origin은 `http://localhost:5173`이다.
- `CORS_ALLOWED_ORIGINS`는 쉼표로 여러 origin을 받을 수 있다.
- origin의 공백, 빈 항목 및 끝 slash를 정규화한다.
- wildcard origin을 허용하지 않는다.
- credentials를 허용하지 않는다.
- HTTP 메서드는 `GET`, `POST`로 제한한다.
- 요청 헤더는 `Accept`, `Content-Type`으로 제한한다.
- 운영 환경에서는 실제 배포 origin만 명시적으로 설정한다.
