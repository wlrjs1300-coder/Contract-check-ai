## 알려진 위험

- 실제 OCR 품질·한국어·표 인식은 검증되지 않음
- Thread timeout은 실행 중 작업을 process 수준에서 강제 종료하지 않음
- Pillow decoder가 OS-level sandbox 안에서 실행되지 않음
- 낮은 해상도·흐림·노출 평가는 초기 heuristic이며 정확성 판정이 아님
- 기존 DB에는 신규 JSON column migration이 필요할 수 있으나 migration은 이번 범위에 없음
