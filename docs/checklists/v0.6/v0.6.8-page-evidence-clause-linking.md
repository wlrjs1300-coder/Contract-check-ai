# v0.6.8 페이지 근거·조항 연결 체크리스트

## 완료 기준

- evidence 매핑 생성 함수 분리 (`evidence_linking`).
- quote 매칭 상태 `exact_match`, `normalized_unique_match`, `ambiguous_match`, `no_match`.
- 스냅샷 해시 불일치 시 `snapshot_stale` 판정 가능.
- 조항 splitter의 fallback 동작과 marker split 동작이 확인됨.
- multi-page 조항에서 `page_start`, `page_end` 보존.
- 증거 검증 실패 시 분석 작업 전체 실패 처리(atomic validation).

## 보안

- source_text는 길이/길이 초과 시 예외 처리.
- block id/페이지/오프셋은 스냅샷 내값만 사용.
- 합성 데이터/fixture만 회귀 테스트에 사용.
