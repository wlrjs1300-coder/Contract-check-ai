# ADR 템플릿

ADR은 Architecture Decision Record의 약자로, 프로젝트에서 중요한 기술적/기획적 결정을 기록하는 문서이다.

파일명 예시:

```text
docs/decisions/ADR-001-project-direction.md
docs/decisions/ADR-002-tech-stack.md
docs/decisions/ADR-003-contract-scope.md
docs/decisions/ADR-004-privacy-first-design.md
```
