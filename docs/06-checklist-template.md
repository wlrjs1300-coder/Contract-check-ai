## 검증 체크리스트

### 공통 검증

- [ ] `git status` 확인
- [ ] `git diff --cached` 확인
- [ ] `git diff --cached --name-only` 확인
- [ ] 변경 파일 확인
- [ ] 공개 저장소에 올려도 되는 내용인지 확인
- [ ] API Key, 토큰, 계정 정보 등 민감정보가 포함되지 않았는지 확인
- [ ] 원본 계약서 파일이 포함되지 않았는지 확인
- [ ] 작업 중간 산출물 또는 외부 도구 입력/출력 원문이 포함되지 않았는지 확인
- [ ] 자동 삽입된 공동 작성자 문구가 포함되지 않았는지 확인

### 문서 작업인 경우

- [ ] 문서 내용 확인
- [ ] 파일 경로 확인
- [ ] 표현이 서비스 포지션과 일치하는지 확인
- [ ] “법률 자문”으로 오해될 표현이 없는지 확인

### 기능 작업인 경우

- [ ] 로컬 실행 확인
- [ ] 프론트엔드 lint 확인
- [ ] 프론트엔드 build 확인
- [ ] 백엔드 테스트 확인
- [ ] 주요 기능 수동 확인
- [ ] 에러 상태 확인
- [ ] 데이터 없음 상태 확인
- [ ] 보안 관련 로그 확인
