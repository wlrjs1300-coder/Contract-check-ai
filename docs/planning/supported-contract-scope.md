# ContractCheck AI 지원 계약서 범위

## 1. 문서 목적

이 문서는 v0.1.3 독립 기획 산출물이다.

v0.1.2에서 정리한 MVP 범위 중 계약서별 지원 범위를 구체화하고, 근로계약서와 주택 임대차계약서의 지원, 선택, 제한, 제외 범위를 정의한다.

본 문서는 법률 판단이나 분석 규칙 상세를 확정하지 않는다. 조항별 법률 기준, 위험도 산정, AI 프롬프트, 응답 구조는 후속 단계에서 다룬다.

## 2. 지원 계약서

MVP 공식 지원 계약서는 다음 2종이다.

- 근로계약서
- 주택 임대차계약서

다음 계약서 유형은 MVP 공식 지원 범위에서 제외한다.

- 상가 임대차계약서
- 프리랜서 용역계약서
- 외주개발 계약서
- 매매계약서
- 대출계약서
- 투자계약서
- 주주간계약서
- 동업계약서
- 비밀유지계약서 단독 문서
- 보험계약서
- 이용약관
- 전자상거래 계약
- 가족 간 계약
- 국제계약

상가 임대차계약서, 프리랜서 용역계약서, 외주개발 계약서, 비밀유지계약서 단독 문서는 2차 확장 후보로 분리할 수 있다.

## 3. 지원 깊이 원칙

- 근로계약서와 주택 임대차계약서는 핵심 조항 중심으로 동일한 깊이로 지원한다.
- 두 계약서 중 하나를 우선하거나 더 깊게 지원하지 않는다.
- 계약서 전체의 적법성이나 유효성을 판정하지 않는다.
- 조항 존재 여부, 누락 가능성, 모호성, 불리할 가능성을 안내한다.
- 세부 법률 기준과 분석 규칙은 v0.5 또는 별도 법률 검증에서 다룬다.
- 계약서에 없는 외부 사실을 추정하지 않는다.

## 4. 범위 분류 기준

| 분류 | 의미 |
|---|---|
| 필수 지원 | MVP에서 계약서별 핵심 조항으로 확인해야 하는 항목 |
| 선택 지원 | MVP 일정과 구현 범위에 따라 포함할 수 있는 항목 |
| 제한 지원 | 계약서 내부 문구 존재 여부와 별도 확인 필요성만 안내하는 항목 |
| MVP 제외 | 이번 MVP에서 공식 지원하지 않는 계약서 유형 또는 기능 |

제한 지원은 계약서 내부 문구 확인까지만 다룬다. 외부 정보 조회, 외부 서류 검증, 법적 효력 판단, 요건 충족 판단은 포함하지 않는다.

## 6. 근로계약서 선택 지원 범위

| 항목 | MVP 확인 범위 | 지원 경계 |
|---|---|---|
| 해고 관련 문구 | 해고 또는 계약 종료 관련 문구 존재 여부 확인 | 해고의 적법성이나 무효 여부를 판단하지 않는다. |
| 퇴직 관련 문구 | 퇴직, 퇴직금, 인수인계 관련 문구 존재 여부 확인 | 구체 권리 산정이나 법률 판단은 하지 않는다. |
| 비밀유지 | 비밀유지 의무 문구 존재 여부와 부담 가능성 확인 | 의무의 법적 효력은 판단하지 않는다. |
| 경업금지 | 경업금지 또는 유사 제한 문구 존재 여부 확인 | 효력과 적법성은 판단하지 않고 전문가 검토를 권고할 수 있다. |

선택 지원 항목은 문구 존재와 부담 가능성을 표시하는 범위로 한정한다. MVP 일정에 따라 구현에서 제외될 수 있다.

## 8. 주택 임대차계약서 제한 지원 범위

| 항목 | MVP 확인 범위 | 지원 경계 |
|---|---|---|
| 계약갱신 관련 문구 | 계약갱신 또는 연장 관련 문구 존재 여부 확인 | 갱신 권리나 법적 요건 충족 여부를 판단하지 않는다. |
| 권리관계 확인 필요 여부 | 계약서 내부의 권리관계 관련 안내 또는 확인 문구 존재 여부 확인 | 등기부등본 등 외부 서류를 조회하거나 판단하지 않는다. |
| 선순위 권리 확인 필요 여부 | 선순위 권리 관련 문구 또는 별도 확인 필요성 안내 | 선순위 권리 존재 여부나 금액을 판정하지 않는다. |
| 확정일자 확인 필요 여부 | 확정일자 관련 문구 또는 확인 필요성 안내 | 확정일자 효력이나 요건 충족 여부를 판단하지 않는다. |
| 전입신고 확인 필요 여부 | 전입신고 관련 문구 또는 확인 필요성 안내 | 대항력이나 우선변제권 충족 여부를 판단하지 않는다. |

제한 지원 항목은 계약서 문구 또는 관련 안내 존재 여부만 확인한다. 외부 서류 확인, 요건 충족 판단, 법적 효력 판단은 제외하고 사용자에게 별도 확인 필요성을 안내한다.

## 9. 주택 임대차계약서 선택 지원 범위

| 항목 | MVP 확인 범위 | 지원 경계 |
|---|---|---|
| 전대 금지 | 전대 금지 또는 제한 문구 존재 여부 확인 | 위반 여부나 법적 효력은 판단하지 않는다. |
| 반려동물 관련 조건 | 반려동물 허용, 제한, 원상복구 관련 문구 확인 | 분쟁 가능성 안내 수준으로 한정한다. |
| 보증보험 관련 안내 | 보증보험 가입 또는 확인 관련 문구 존재 여부 확인 | 가입 가능성, 보장 범위, 외부 절차 판단은 하지 않는다. |

보증보험은 계약서 분석보다 외부 절차 성격이 강하므로 선택 지원 또는 2차 확장 후보로 둔다.

| 책임 범위 | 근로계약서, 주택 임대차계약서 | 포함 | 책임 주체와 범위가 불명확할 수 있다. |
| 자동 갱신 가능성 | 근로계약서, 주택 임대차계약서 | 선택 | 계약서 유형별 차이가 있어 확인 필요성 안내로 제한한다. |
| 일방 변경 권한 | 근로계약서, 주택 임대차계약서 | 포함 | 한쪽 당사자에게 넓은 변경 권한이 있는지 확인한다. |
| 모호한 표현 | 근로계약서, 주택 임대차계약서 | 포함 | 사용자 이해와 추가 확인 질문 정리에 필요하다. |
| 누락 가능성이 있는 핵심 정보 | 근로계약서, 주택 임대차계약서 | 포함 | 핵심 조건이 빠졌을 가능성을 안내한다. |
| 불리할 가능성이 있는 특약 | 근로계약서, 주택 임대차계약서 | 포함 | 부담 집중 가능성을 확인한다. |
| 전문가 검토 권고 필요성 | 근로계약서, 주택 임대차계약서 | 포함 | 법률 판단을 대신하지 않고 추가 검토 필요성을 안내한다. |

위 항목은 법률적 결론이 아니라 확인 필요성을 안내하는 범위로 작성한다.

## 11. 특약사항 처리 범위

특약사항은 다음 범위에서만 확인한다.

- 문구가 모호한지
- 금액, 기간, 의무, 책임 범위가 불명확한지
- 한쪽 당사자에게 부담이 집중될 가능성이 있는지
- 본문 조항과 충돌할 가능성이 있는지
- 전문가 확인이 필요한지

다음은 포함하지 않는다.

- 특약 위험도 점수 확정
- 특약 유형별 위법성 판정
- 특약 조항별 법률 기준 수치 확정
- 모든 특약 자동 분류
- 법적 효력 확정
- 특약 수정 문안 자동 작성

## 12. 위험 분석 표현 경계

| 구분 | 표현 | 사용 기준 |
|---|---|---|
| 허용 | 확인 필요 | 사용자가 추가로 확인해야 할 항목에 사용한다. |
| 허용 | 누락 가능성 | 핵심 정보가 문서에 없거나 찾기 어려울 때 사용한다. |
| 허용 | 불리할 가능성 | 부담이 한쪽에 집중될 수 있는 문구에 사용한다. |
| 허용 | 모호한 표현 | 의미나 기준이 명확하지 않은 문구에 사용한다. |
| 허용 | 추가 확인 권장 | 사용자 확인이나 전문가 상담 전 정리가 필요한 경우 사용한다. |
| 허용 | 전문가 검토 권고 | 법률 판단을 대신하지 않고 별도 검토 필요성을 안내할 때 사용한다. |
| 주의 | 위험 조항 | 단정적 결론이 아니라 확인 우선순위 의미로만 사용한다. |
| 주의 | 일반적인 계약 관행과 비교 필요 | 구체 기준을 확정하지 않고 비교 필요성만 안내한다. |
| 금지 | 불법 | 위법성을 단정하므로 사용하지 않는다. |
| 금지 | 위법 | 서비스가 조항의 위법 여부를 확정하는 것으로 오인될 수 있으므로 사용하지 않는다. |
| 금지 | 무효 | 법적 효력을 단정하므로 사용하지 않는다. |
| 금지 | 적법성 확정 | 계약서 전체 또는 개별 조항의 적법성을 최종 판단하는 표현이므로 사용하지 않는다. |
| 금지 | 사기 | 범죄나 기망을 단정하므로 사용하지 않는다. |
| 금지 | 법적으로 안전 | 안전성을 보장하므로 사용하지 않는다. |
| 금지 | 법적 효력 보장 | 계약서나 조항의 법적 효력을 보장한다고 오인될 수 있으므로 사용하지 않는다. |
| 금지 | 반환 보장 | 보증금 등 금전 반환을 보장하므로 사용하지 않는다. |
| 금지 | 승소 보장 | 소송 결과를 보장하므로 사용하지 않는다. |
| 금지 | 해고 무효 확정 | 법적 결론을 단정하므로 사용하지 않는다. |
| 금지 | 계약 효력 없음 | 계약 효력을 단정하므로 사용하지 않는다. |

## 13. 외부 정보 및 기능 제외

MVP에서는 다음 기능을 포함하지 않는다.

- 등기부등본 자동 조회
- 공공기관 데이터 조회
- 선순위 권리 자동 판정
- 보증금 안전성 계산
- 법적 요건 충족 여부 판정
- 변호사 자동 연결
- 계약서 자동 수정

## 14. 계약서 유형 판별 경계

- 사용자가 선택한 계약서 유형과 추출된 문서 내용이 일치하는지 확인할 필요가 있다.
- 계약서 유형을 확정하는 구체 규칙은 v0.5에서 다룬다.
- 혼합형 또는 비정형 계약서는 지원이 제한될 수 있다.
- 다른 계약서 유형을 잘못 분석하지 않도록 지원 불가 안내가 필요하다.

상세 분류 알고리즘은 이번 문서에서 작성하지 않는다.

## 15. 완료 기준

- 공식 지원 계약서 2종이 명시되어 있다.
- 계약서별 필수, 선택, 제한 범위가 구분되어 있다.
- 공통 분석 항목이 정리되어 있다.
- 외부 정보 조회 범위가 제외되어 있다.
- 법률적 판단과 확인 필요 안내가 구분되어 있다.
- MVP 제외 계약서가 명시되어 있다.
- 후속 설계 경계가 명확하다.
