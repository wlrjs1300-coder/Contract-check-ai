# v0.2.1 Fixture Rules

This directory contains only small, tracked, non-sensitive fixture files for v0.2.1 spikes.

## Allowed

- Fully synthetic text
- Small `.sample.txt` files
- Fictional identifiers such as `가상 사용자 001`
- Fictional company labels such as `예시회사 001`
- Fictional addresses such as `예시시 예시구 예시로 100`
- Invalid reserved email domains such as `example.invalid`

## Not Allowed

- Real contract text
- Real contract files
- Real names
- Real companies
- Real addresses
- Valid phone numbers
- Resident registration number-like strings
- Account number-like strings
- API keys, secrets, tokens, passwords
- Raw external responses
- Product analysis results

Fixtures are not legal templates and must not be presented as valid contracts.
